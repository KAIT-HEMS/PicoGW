PicoGW

A minimalist's home gateway that supports ECHONET Lite. The license is MIT.
The primary distribution site is here.
To access ECHONET Lite device, right click the device's property name (such as OperatingState) and select 'Open API', which generates ECHONET Lite's get access to the property in new tab. If you want set a new value to a property, you access the same URL by PUT method with the desired parameter value. For example:
http://[IP address]:8080/v1/echonet/AirConditioner_1/OperatingState/?30
will set 0x30 to OperatingState property (meaning Power ON). If your browser does not support PUT access, you can also mimic it by adding 'PUT' at the end of GET URL. For example:
http://[IP address]:8080/v1/echonet/AirConditioner_1/OperatingState/PUT?30
